Вопросы

Коротко про домен, оплату и правки.

Нужно ли платить за хостинг?

Для простого сайта можно использовать бесплатное размещение на GitHub Pages. Домен оплачивается отдельно, только если нужен свой красивый адрес.

Что входит в цену?

Структура, дизайн, верстка, мобильная версия, публикация, базовые правки и инструкция по дальнейшему использованию.

Как проходит оплата?

Оплата делится на этапы после согласования задачи. Способ оплаты и документы обсуждаются заранее, чтобы обе стороны понимали условия.
